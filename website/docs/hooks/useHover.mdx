## API

```typescript
const hover = useHover(ref, 5000);
```

### Params

| 名称  | 说明           | 类型                    |
| :---- | :------------- | :---------------------- |
| ref   | 目标元素       | RefObject<HTMLElement\> |
| delay | 延时时长(毫秒) | number                  |

### Result

| 名称  | 说明            | 类型    |
| :---- | :-------------- | :------ |
| hover | 元素 hover 状态 | boolean |
