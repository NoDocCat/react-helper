使用设备电池状态

:::caution 警告
此 hook 依赖非标准的 API, 且这组 API 随时有可能被移除, 请务必在使用时正确检查 isSupport 状态.
即使在受支持的环境中, chargingTime 与 dischargingTime 受限于系统限制, 状态并非完全可靠.
:::

## API

### Result

| Name            | Type    | Description              |
| --------------- | ------- | ------------------------ |
| isSupport       | boolean | 当前环境是否支持电池 API |
| charging        | boolean | 充电状态                 |
| chargingTime    | number  | 距离充电完毕还需多少秒   |
| dischargingTime | number  | 距离电池耗尽还需多少秒   |
| level           | number  | 电量放大等级             |
